Overview

Darvis is a voice assistant that provides both a desktop GUI and web interface. It supports multiple users with isolated sessions and real-time AI-powered conversations.

Key Features

  • Voice activation ("Hey Darvis" / "Computer")
  • Text input via web UI
  • Multi-user support with session isolation
  • Real-time message updates via Socket.IO
  • AI responses via OpenCode CLI
  • Desktop app launching
  • Waybar status integration (Linux)

Socket Room Isolation

Each user is assigned to a room named user_{user_id}. All messages are emitted only to the user's room:

socketio.emit('ai_message', {'message': response, 'session_id': session_id}, to=f"user_{user_id}")
