Quick Installation

Get Darvis up and running in minutes

1

Clone Repository

git clone https://github.com/darobbins85/darvis.git
cd darvis
2

Setup Environment

python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
3

Install Dependencies

pip install -r requirements.txt
pip install -r requirements-dev.txt
4

Launch Darvis

./launch-darvis.sh

Platform Requirements

  • Python 3.7+
  • Microphone access
  • Speakers/audio output

Optional Extras

  • Linux: PyGObject for system tray
  • Desktop integration support
  • Waybar for Linux status display

Pro Tip

For desktop integration on Linux, run ./install-desktop.sh to add Darvis to your application menu with system icon.

Voice Commands

Simple, natural voice interactions

Application Control

"hey darvis, open calculator" Launches system calculator
"hey darvis, open firefox" Opens Firefox browser
"hey darvis, launch terminal" Opens terminal application

AI Queries

"what is 2 + 2" Mathematical calculations
"what time is it" Current time and date
"tell me a joke" Entertainment and fun

System Control

"hey darvis, volume up" Increase system volume
"hey darvis, mute" Mute/unmute audio
"hey darvis, shutdown" System shutdown (with confirmation)
